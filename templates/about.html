What is MetaTorrent?

MetaTorrent is personal project made to learn some technlogies like flask or requets. Is a simple micro-project create to try some things.

Is a serious project?

No. Only is a experimental project, its operation is very poor, but are invited to collaborate. Feel free to use or improve the code.

How it works?

MetaTorrent is just a meta search engine, this means it use other search engine and not log or store anything. To realize a search only write in the search box the keywords and press the search button. If you can take the necessary time the results will appear below with a link to the original search engine, the magnet link and other information.

Is possible sort the results?

No. This feature is not yet implemented.

Remember. It's a experimental project without support
